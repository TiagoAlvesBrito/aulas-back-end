Para gerar o esqueleto dessa API pelo express:

npm install express-generator -g

express -h (Mostra as opções disponíveis) - (Adicionar o npm nas variaveis de ambiente se não funcionar)

express --no-view nomedaminhaapi

instruções: https://expressjs.com/pt-br/starter/generator.html#gerador-de-aplicativos-do-express
